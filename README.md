# DRF-FullStackApp-Spotify

Simple app based on DRF + React and Integrated with expernal API

To run:
- Backend: <br>
`pip install -r requirements.txt`          
`python manage.py runserver`
<br>
- Frontend: <br>
`npm run dev`
