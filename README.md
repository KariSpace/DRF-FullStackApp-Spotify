# DRF-FullStackApp-Spotify

Simple app based on DRF + React and Integrated with expernal API

Ready: 
- Django App, 
- Models, 
- Basic vievs, 
- Simple react app integrated with DRF

To run:
- Backend: 
pip install -r requirements.txt           
python manage.py runserver 

- Frontend: 
npm run dev
